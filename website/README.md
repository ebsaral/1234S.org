# Logical Spirituality Web App

[Next.js](https://nextjs.org) web application of [1234s.org](https://1234s.org)


## Setup (local)

First, install packages:
```bash
npm install
# or
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Inter, Playfair Display fonts](https://vercel.com/font).
